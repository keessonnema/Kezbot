# Kezbot (Shifty)

[![Telegram Bot](https://img.shields.io/badge/Telegram-Bot-blue.svg)](https://t.me/Kezbot)
[![Python](https://img.shields.io/badge/Python-3.x-green.svg)](https://www.python.org)

> [!WARNING]
> **Project Status: Unmaintained**
> This repository is currently not actively maintained and has not been updated for several years. While it remains popular and the codebase serves as a solid reference, please note that some dependencies or API integrations may be outdated or require manual adjustments to run.

**Kezbot** (known as [@Kezbot / Shifty](https://t.me/Kezbot) on Telegram) is a modular, open-source Python Telegram bot designed to bridge the gap between streaming platforms. Send it a link from YouTube or Spotify, and Kezbot will instantly find and return the equivalent link on the alternative platform. 

---

## 🚀 Features

* **Cross-Platform Link Conversion:**
    * Convert **YouTube** URLs to **Spotify** tracks instantly.
    * Convert **Spotify** URLs to **YouTube** videos/audio instantly.
* **Playlist Search:** Search for a term and receive a curated list of the top 10 relevant playlists.
* **Utility Tools:** Quickly retrieve your unique Telegram User ID.
* **Modular Architecture:** Built with extensibility in mind, making it easy to plug in new modules and platforms.

---

## 🛑 Anti-Piracy Policy

> [!IMPORTANT]
> This project is **strictly against piracy and illegal music downloading**. Kezbot only converts links between legitimate streaming services. 
> 
> Any pull requests or issues introducing features that facilitate downloading, scraping, or bypassing paywalls of copyrighted material **will be immediately rejected and ignored**, and contributors violating this policy may be blocked from future contributions.
